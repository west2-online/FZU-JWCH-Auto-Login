>本脚本适用于FZU教务处本科教学管理系统的验证登录。

首次使用请在悬浮页面输入**学号**和**密码**，并点击保存。

本脚本**不会上传学号和密码**，这些数据均储存在本地。

如果对这个脚本有更好的建议或者使用中产生问题，欢迎反馈

### 移动端支持

- X浏览器最新版可用（登录跳转会被识别为广告跳转，请设置白名单）
- 其他移动端浏览器可能不兼容此脚本，请自测。
